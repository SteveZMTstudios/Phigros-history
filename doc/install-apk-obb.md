# xapk 和 apk+obb 安装指南

您拿到的包长什么样呢？
- 后缀为`.7z`，里面有base.apk和`xxx.obb`文件 --> [是`APK+OBB`格式](#apkobb-简单安装指南)
- 后缀为`.xapk` --> [是`XAPK`格式](#xapk-简单安装指南)
- 后缀为`.001` `.002`等分卷压缩包 --> 请参考[分卷压缩包解压指南](unzip-parsed-file.md)进行解压


## APK+OBB 简单安装指南
### 准备工作
- 手机上准备好任一文件管理器，要能够访问手机目录，如[MT管理器](https://www.coolapk.com/apk/bin.mt.plus)
- 下载好`apk`和`obb`两个包。

### 安装
1. 直接打开`apk`文件安装好,安装好后先不要打开。
2. 找到.obb文件，将它移动到`/storage/emulated/0/Android/obb/com.PigeonGames.Phigros/`目录下。
3. 打开游戏。

## XAPK 简单安装指南
### 准备工作
- 手机上准备好任一文件管理器，要能够访问手机目录，如[MT管理器](https://www.coolapk.com/apk/bin.mt.plus)
- 下载好`xapk`包。

### 安装（MT管理器）
1. 打开MT管理器，找到下载好的`xapk`文件，点击它。
2. MT管理器会自动识别`xapk`文件，并弹出安装界面，点击安装
3. 授予MT管理器安装外部应用的权限（如果有提示的话）
4. 等待安装完成。

### 安装（通用）
1. 找到`xapk`文件，将它改名为`.zip`后缀名，然后解压缩。
2. 进入解压后的文件夹，找到`apk`和`obb`两个文件。
3. 直接打开`apk`文件安装好,安装好后先不要打开。 
4. 找到obb文件，将它移动到`/storage/emulated/0/Android/obb/com.PigeonGames.Phigros/`目录下。
5. 打开游戏。
